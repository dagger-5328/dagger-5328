## 🧠 Featured Project

### Crypto Pulse – LSTM Signal Prediction

Predicting cryptocurrency price movement is highly noisy and biased due to market volatility.

- Built LSTM-based time-series model using rolling window sequences  
- Applied walk-forward validation with purging to prevent data leakage  
- Achieved **ROC-AUC ~0.75 (vs ~0.52 baseline)**
